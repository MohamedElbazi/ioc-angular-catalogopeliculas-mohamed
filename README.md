# ioc-angular-catalogopeliculas-mohamed - Mohamed El Bazi

## Descripció
Aquesta aplicació és un **catàleg de pel·lícules**. Permet visualitzar una llista de pel·lícules, amb detalls bàsics de cada una, i està preparada per afegir funcionalitats futures com components reutilitzables, serveis de dades i pàgines principals.

## Stack Tècnic
- Angular 18 (Standalone)
- SCSS (Sass)
- Routing activat (`app.routes.ts`)
- SSR desactivat (SPA)

## Estat
Configuració inicial completada i verificada.
- Projecte generat amb Angular CLI 18.
- Estructura de carpetes `components/`, `services/`, `models/`, `pages/`.
- Ramas preparades i pujades a GitHub (`main`, `ra1-setup`, `ra2-components`, `ra3-serveis`, `ra4-navegacio`).
- Routing activat i hot reload funcionant.
